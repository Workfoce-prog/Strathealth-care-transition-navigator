# Model artifacts

This folder contains working machine-learning artifacts for the Streamlit demonstration.

- `care_transition_model.pkl` — selected model (Logistic Regression)
- `logistic_baseline.pkl` — transparent baseline
- `random_forest_model.pkl` — nonlinear comparison model
- `predict.py` — reusable prediction functions
- `train_model.py` — reproducible training script
- `model_metrics.json` — holdout metrics and metadata
- `feature_importance.csv` — permutation importance on the synthetic holdout set

All records and outcomes are synthetic. Metrics demonstrate software functionality only and must not be interpreted as clinical evidence.
